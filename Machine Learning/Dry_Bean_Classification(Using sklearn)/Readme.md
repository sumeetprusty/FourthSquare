# Dry Bean Classification

This project aims to classify different types of dry beans using a RandomForestClassifier. The dataset used for this project is the Dry Bean Dataset.

## Dataset

The dataset contains several features that describe various properties of dry beans. The target variable is `Class`, which represents the type of dry bean.

## Requirements

The following Python packages are required to run the code:

- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn

You can install the required packages using the following command:

pip install pandas numpy matplotlib seaborn scikit-learn
